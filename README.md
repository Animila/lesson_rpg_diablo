## <a id="list">3. listsPlace<a/>
### Класс Place(Village, Scale)
Класс для управления переходами между локациями. Наследует все из класс локаций
### Функции
#### location(user)
- принимает на вход объект пользователя
- Выводит список локаций
- Производит процесс перехода на локации


### Класс Village
Локация деревни.
В конструкторе вызывается функция select()
### Поля
- Объект пользователя (user)
- Список оружия (armor)
- Список предметов для торговца (goods)
### Функции
#### select()
- Реализует переход внутри локации
#### forge()
- Кузница
- Вывод списка предметов
- Реализация системы выбора предмета и передачи предмета игроку
#### market()
- Рынок
- Вывод списка предметов
- Реализация системы выбора предмета и передачи предмета игроку


### Класс Scale
Локация скалы.
В конструкторе вызывается функция битвы()
### Поля
- Объект пользователя (user)
- Список действий персонажа (action_list)
### Функции
#### war()
- Создание объекта демона
- Реализация системы вывода статистики обоих персонажей
- Реализация системы выбора оружия
- Реализация системы атаки
- Реализация системы завершения битвы

- Принимает на вход любое число
- Реализует систему восстановления здоровья
#### drink_mana_potion(potion_mana)

## <a id="hero">6.4.2. Pattern/person/Hero_pattern <a/>
### Класс Hero(Character, Package)
Шаблон для героев. Наследуется от [Character](#character_pattern) и [Package](#package)

### Поля 
- Сила (thisPower)
- Сила удара (thisKnack)
- Сила интеллекта (thisIntel)
- Опыт (thisExp)
- Уровень (thisLevel)
- Объект сумки (Pack)
### Функции
#### scream()
- Приветственное сообщение
#### attack(target)
- Принимает на вход объект для атаки
- Вывод сумки

- система атаки
